# Election Guide AI — Education Assistant

An interactive, bilingual (English/Hindi) educational platform designed to simplify the Indian election process. Features an offline-ready AI Chat Assistant, a dynamic 40-question quiz bank, and a comprehensive guide to voting rights, EVM technology, and electoral procedures.

## 🚀 Features
- **Intelligent Chat Assistant**: Over 60 intents covering Voter Registration, EVMs, VVPATs, and Electoral Laws.
- **Interactive Quiz**: Randomized 5-question sessions from a 40-question bank with detailed bilingual explanations.
- **Bilingual Support**: Full support for both English and Hindi.
- **Offline First**: No external APIs required. Works entirely with local data for speed and privacy.
- **Glassmorphism UI**: Modern, premium design with dark/light mode toggle.

## 🛠️ Technology Stack
- **Framework**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **Deployment**: Static Site ready (Vercel/Netlify/GitHub Pages)

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Bhavesh-png/Election-Process-Education-Assistant.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
